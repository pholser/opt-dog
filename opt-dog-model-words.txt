The duration of a breed judging event e is deemed to be:
-- RATE[EJ[e]] * sum[x in X, d in D] { N[d, x, e] }

The duration of a group breed judging event e for group g is
deemed to be:
-- RATE[EJ[e]]
   *
   (number of breed events for breeds in group g)

The duration of a best-in-show breed judging event e is:
-- RATE[EJ[e]]
   *
   (number of group events)

We want to schedule the breed judging events in blocks of time,
each not to exceed some maximum, say, an hour. Every block's events
are judged by the same judge. The hour-ish maximum provides natural
breaks in time. Longer blocks would leave exhibitors later on in
the ring less certain about their start times. Then each block is
assigned to a start time in a ring.

We want to start blocks of judge time on a five-minute boundary.

The events in a block, however they are ordered, are considered to run
back-to-back. A block's start time thus induces the start times of the
events in the block, by virtue of ordering and the estimated durations of
the events.

We will thus need to decide on a partition of each judge's events into
one or more blocks, the ordering of the events within each blocks,
and the ring and start time for each block.

Practically speaking, the number of possible block configurations is
too large to have the model decide on. Suppose we start with a particular
block configuration as fixed, and assert that the blocks already will
obey the required constraints:
  - every event occupies one position in one block
  - duration of <= 60 min unless an entry in a breed/variety exceeds 30
  - all events in a block are judged by the same judge
We can find the best solution given this one block configuration, and
repeat for as many feasible block combinations as we wish.

Let K = {k_1, k_2, ..., k_{nk}} be the blocks constructed to obey the
block constraints.
Let EK_k = an ordered list of the events in block k, for all k in K.

Let T = {1, 2, ..., MAX_T} be the possible start times for a block.
Each member of T corresponds to a five-minute marker: 8:00am, 8:05am,
etc.

   be scheduled at the earliest possible opportunity.
   Per the Rules Applying to Dog Shows, a group start time
   must be printed in the judging program. Scheduling the
   start time at the earliest possible time allows clubs
   to adjust as the day goes on; procedures allow for
   scheduled times to be delayed but may only start
   earlier than the scheduled time based on extenuating
   circumstances."
   ...Minimize the maximum end time of a judging block.

  "Attempt to schedule judges in a single ring. If a judge

   relocate the judge during their lunch break."

     during breed events.
